# toMarkdown

Program that reformats code files automatically generated by Google Colab/Jupyter Notebook to markdown with a modern GUI interface.

<img width="1403" height="896" alt="image" src="https://github.com/user-attachments/assets/3143d00c-b0c6-430f-af53-ab311c9684db" />

## Purpose

If you frequently work with Google Colab or Jupyter Notebooks and also use Obsidian or Markdown for documentation, this tool provides a seamless bridge between the two. While Colab offers many conveniences for interactive computing, it does not natively support exporting notebooks to Markdown - a format that excels for structured documentation and knowledge management.

This tool was created out of a passion for high-quality documentation and the need to preserve both the narrative and technical content of notebooks. It is especially valuable in academic and research settings, where Colab notebooks often contain extensive annotations, insights, and runnable examples alongside code.

With a clean, intuitive GUI and just a few clicks, you can now convert auto-generated .py files from Colab into well-organized, readable Markdown. Whether youâ€™re curating notes, preparing tutorials, or maintaining a personal knowledge base, this tool makes the transition effortless and efficient.

## How it works

**Input:**

```python
"""
# My Project
This is my documentation
"""

import pandas as pd

"""
## Section 1
Some explanation here
"""

def my_function():
    pass
```

**Output:**

````md
# My Project 
This is my documentation
```python 
 import pandas as pd 
``` 
## Section 1 
Some explanation here 
```python 
def my_function(): pass 
```
````
## Requirements

- **Java JDK 11+**

```bash
javac --version
```

## Installation

### MacOS/Linux

```bash
# Clone the repo
git clone https://github.com/bogdan-tr/Colab_to_markdown.git
# Enter the directory
cd Colab_to_markdown
# Make it executable
chmod +x install.sh
# Run the install script
./install.sh
```
### Windows

```bash
# Clone the repo
git clone https://github.com/bogdan-tr/Colab_to_markdown.git
# Enter the directory
cd Colab_to_markdown
# Run install script
bash install.sh
```

That's it! Now run it with `tomd` in the terminal.
## Uninstall

```bash
cd Colab_to_markdown
chmod +x uninstall.sh
./uninstall.sh
```
